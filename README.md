## Hi, I'm Rohan

**ML Lead** working at the intersection of **large language models** and **healthcare**.

I build agentic AI systems that make clinical data useful — from querying 1,200-table data warehouses to extracting structured knowledge from messy clinical text. I care about shipping things that work in production, not just in notebooks.
